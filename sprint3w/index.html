    Documentació: Configuració i gestió de dominis Windows Server

    1. Implementació i configuració del domini

    1.1 Teoria

    Active Directory Domain Services (AD DS) permet gestionar usuaris, equips, grups i recursos dins d’un domini Windows.

    1.2 Procediment

    1. Instal·lar el rol Active Directory Domain Services des de Server Manager.

    1. Promocionar el servidor a controlador de domini:
    2. Crear un nou bosc (forest) amb nom com empresa.local.

    • Assignar una contrasenya DSRM.

    • Esperar que finalitzi la configuració i reinici del sistema.

    2. Gestió de comptes d'usuari i permisos

    2.1 Creació d'usuaris

    1. Obrir Active Directory Users and Computers (ADUC).

    1. Crear usuaris a una Organizational Unit (OU).

    1. Establir polítiques de contrasenya i dates d'expiració.

    2.2 Assignació de permisos

    • Crear carpeta compartida (D:\Dades\Usuaris).

    • Configurar permisos NTFS i permisos de compartició.

    • Aplicar Llistes de Control d'Accés (ACL) a usuaris o grups.

    3. Perfils mòbils i accés remot a la informació

    3.1 Teoria

    Els perfils mòbils permeten carregar configuracions personals des de qualsevol dispositiu del domini.

    3.2 Configuració

    1. Crear carpeta compartida (\\server\perfils$) amb permisos adequats.
    2. A ADUC: Propietats de l’usuari > Profile Path:
      \\server\perfils$\%username%

    4. Gestió de grups de seguretat

    4.1 Creació i ús

    1. Crear grups de seguretat a ADUC.

    1. Afegir usuaris als grups.

    5. Accés a recursos locals i en xarxa

    5.1 Recursos locals

    • Compartir carpetes amb permisos adequats.

    • Aplicar seguretat NTFS per restringir o permetre accés.

    5.2 Recursos en xarxa

    • Servidors de fitxers: compartir directoris centrals.
    • Servidors d’impressió: instal·lar, compartir i assignar impressores.
    • Servidors d’aplicacions: ex. IIS, SQL Server, aplicacions d’empresa.

    6. Directives de seguretat i drets d'usuari

    6.1 Ús de GPO (Group Policy Objects)

    1. Obrir Group Policy Management Console (GPMC).
    2. Crear i vincular GPO a una OU.
    3. Configurar:
    4. Polítiques de contrasenya.
    5. Restriccions d’accés.
    6. Configuració d’escriptori, redirecció de carpetes...

    7. Connexió remota i administració de servidors

    7.1 Activació de Remote Desktop (RDP)

    1. A System Properties > Remote, habilitar connexions remotes.
    2. Configurar tallafocs per permetre port 3389.
    3. Utilitzar mstsc des d’un client per accedir al servidor.

    8. Mesures de seguretat bàsiques

    8.1 Protecció del sistema

    • Activar Windows Defender i/o antivirus de tercers.
    • Aplicar polítiques de bloqueig de sessió i auditoria d’accessos.
    • Configurar el tallafocs amb regles estrictes.

    8.2 Eines de seguretat

    • Instal·lació d’antivirus, antispyware, i gestors d’actualitzacions.
    • Restriccions d’ús de dispositius USB si cal.
